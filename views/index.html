Aprendizaje de HTML y CSS: Mi Primer Proyecto de Diseño Web

Como parte de mi proceso de aprendizaje en el mundo del diseño web y la programación, decidí crear este portafolio personal utilizando HTML y CSS puros, dos de las tecnologías fundamentales que he estudiado hasta ahora. Este proyecto marca el comienzo de mi camino como desarrollador web y me permitió aplicar los conocimientos adquiridos de manera práctica. El objetivo de este primer proyecto es presentar un portafolio personal donde no solo comparto mi experiencia profesional, sino también los logros y habilidades que he ido desarrollando en el ámbito laboral. Aquí, el diseño y la estructura se basan en principios de HTML para la organización del contenido y CSS para darle estilo y funcionalidad visual. A lo largo del proceso, utilicé recursos externos que complementan el aprendizaje, y en la siguiente sección del proyecto los detallaré para que pueda seguir mejorando y expandiendo el contenido de este portafolio. Este es solo el inicio de lo que espero sea un camino continuo de aprendizaje y mejora en el desarrollo web.
